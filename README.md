# GenAI Resume Screening System

An AI-powered Resume Screening System built using LangChain and LLMs that evaluates candidates based on a job description.

##  Features
- Skill extraction from resumes
- Matching with job description
- Candidate scoring (0–100)
- Explainable results
- LangSmith tracing for debugging

##  Pipeline
Resume → Extract → Match → Score → Explain

## 🛠 Tech Stack
- Python
- LangChain
- OpenAI / HuggingFace
- LangSmith
